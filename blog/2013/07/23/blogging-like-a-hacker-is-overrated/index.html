Blogging Like a Hacker Is Overrated

by Natasha Callender - Jul 23rd, 2013 -

I am on someone else’s macbook at the moment, and I want to update my blog. My first instinct was to groan, and grumble about Octopress’s portability issues. My second instinct was to think that it is not that bad. All I have to do is,

  1. Install homebrew, git and ruby on this computer
  2. Install Octopress on this computer
  3. Clone my blog
  4. Blog

Ok, I am going to stop here for a second, and cry a little about step 1. I have had a development environment on my mac for so long that I have forgotten what it is like to not have one. Oh crap! I need to install xcode as well. The amended steps for blogging on someone else’s computer are as follows,mo

  1. Sigh heavily
  2. Install Xcode
  3. Install homebrew, git and ruby
  4. Install Octopress
  5. Realise that this is madness and find another solution

I was about to download xcode when the insanity of what I was about to do became clear.

  1. The macbook I am typing on is not mine. I would not be happy if someone downloaded a whole bunch of stuff onto my laptop without permission.
  2. Recreating a dev environment on a laptop I only plan on using for a few hours is a waste of time
  3. The above is a clunky, inelegant, decidedly unsimple (I know that is not a word) solution to my blogging problem.

My compromise solution is to use textedit to write the blog post, which I am doing right now, save it to my Dropbox and upload it when I get back to my computer. I am again struck by the problem with those static blog generators that have to live in a local development environment. With a Dropbox/Markdown blogging solution like scrivr, or calepin, my compromise solution would be the only solution. My process would be to write the post, save it to my dropbox and I am done. The truth is that I do not want to blog like a hacker. I want to blog like a lifehacker.

This feeling is kind of like the day I realized that for keeping track of the things that need keeping track of, a moleskine cashier pocket notebook fit better into my life than my sleek moleskine hard cover calendar, which fit better into my life than any of my tricked out Android devices.
